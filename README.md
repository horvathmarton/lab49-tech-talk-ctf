# lab49-tech-talk-ctf

Material for the Lab49 internal CTF.

## Setup

- Run `npm install`.

## Usage

- Run `npm run build:style` if you change anything in the styling.
- Run `npm run bundle` to create the `static` folder.
- Run `npm run serve` to serve the challenges.
